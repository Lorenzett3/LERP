# LERP Backend

API NestJS do LERP. Expõe rotas para autenticação demonstrativa, painel, produtos, pedidos e movimentos de estoque.

## Rodar em Desenvolvimento

```bash
npm run start:dev
```

A API fica disponível em `http://localhost:3000/api`.

## Build

```bash
npm run build
```

## Testes

```bash
npm run test
```
